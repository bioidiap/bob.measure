bob.measure.regression.mse(estimation, target)[source]¶

Mean square error between a set of outputs and target values

Uses the formula:

\[MSE(\hat{\Theta}) = E[(\hat{\Theta} - \Theta)^2]\]

Estimation (\(\hat{\Theta}\)) and target (\(\Theta\)) are supposed to have 2 dimensions. Different examples are organized as rows while different features in the estimated values or targets are organized as different columns.

Parameters

  • estimation (numpy.ndarray (float)) – A N-dimensional array that corresponds to the value estimated by your procedure

  • target (numpy.ndarray (float)) – A N-dimensional array that corresponds to the expected value

Returns

mse – The average of the squared error between the estimated value and the target

Return type

float

bob.measure.regression.rmse(estimation, target)[source]¶

Calculates the root mean square error between a set of outputs and target

Uses the formula:

\[RMSE(\hat{\Theta}) = \sqrt(E[(\hat{\Theta} - \Theta)^2])\]

Estimation (\(\hat{\Theta}\)) and target (\(\Theta\)) are supposed to have 2 dimensions. Different examples are organized as rows while different features in the estimated values or targets are organized as different columns.

Parameters

  • estimation (numpy.ndarray (float)) – A N-dimensional array that corresponds to the value estimated by your procedure

  • target (numpy.ndarray (float)) – A N-dimensional array that corresponds to the expected value

Returns

rmse – The square-root of the average of the squared error between the estimated value and the target

Return type

float
